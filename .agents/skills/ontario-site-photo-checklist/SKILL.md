## 25. Site Photo Requirement Checklist

### What It Is
A checklist of photographs that should be taken and submitted with a development application to document existing conditions.

### Why It Matters
- Photos are required on most municipal application checklists
- Missing or poor photos can delay applications
- Photos help Committee members understand the site without visiting
- Good photos support the planning rationale narrative

### How to Build It

```
SITE PHOTO REQUIREMENT CHECKLIST

REQUIRED PHOTOS:

SUBJECT PROPERTY:
□ Front elevation of existing building (full view)
□ Side elevations (both sides)
□ Rear elevation
□ Front yard (from street)
□ Rear yard (from building looking back)
□ Side yards (both)
□ Any existing trees on the property
  (with diameter measurement visible)
□ Existing driveway and parking area
□ Any unique features (heritage elements, retaining walls,
  drainage issues, etc.)

STREETSCAPE CONTEXT:
□ Streetscape looking east (from in front of subject property)
□ Streetscape looking west
□ Subject property in context with immediate neighbours
□ Across-the-street view of subject property

ADJACENT PROPERTIES:
□ North adjacent property (front view from street)
□ South adjacent property (front view from street)
□ Rear adjacent property (if visible from lane or yard)
□ Properties across the street

WIDER CONTEXT:
□ Nearest intersection (showing street signs)
□ Nearest transit stop (if within 400m)
□ Nearest park (if within 200m)
□ Any notable features in the area (schools, commercial, etc.)

SPECIFIC TO PROPOSAL TYPE:
If laneway/garden suite:
□ Laneway view (both directions)
□ Width of laneway
□ Other laneway suites or garages visible

If lot severance:
□ Where the proposed lot line would be
□ Both resulting lots (front view)

If tree removal:
□ Each tree to be removed (with tag/identification)
□ Tree canopy spread
□ Root zone area

PHOTO SPECIFICATIONS:
- Format: JPEG or PNG
- Minimum resolution: 2 megapixels
- Date stamp: Photos should be recent (within 3 months)
- Season: Include both leaf-on and leaf-off if possible
  (for shadow and overlook analysis)
- Label each photo with direction and description
- Include a location map showing photo positions and
  direction of view
```
