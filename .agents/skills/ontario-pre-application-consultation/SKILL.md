## 26. Pre-Application Consultation Package

### What It Is
A package of materials prepared for a pre-application consultation meeting with municipal planning staff. This meeting helps identify issues before filing a formal application.

### Why It Matters
- **Required** for most ZBA and OPA applications in Ontario
- Strongly recommended for complex minor variance applications
- Staff provide a **pre-application consultation response** that lists required studies and application materials
- Saves time and money by identifying issues early
- Establishes a working relationship with the assigned planner

### Governing Law
- **Planning Act, s. 22(3.1) and 34(10.0.1)** — Pre-application consultation can be required by municipal by-law
- **O. Reg. 543/06, s. 3.1** — Prescribed information for pre-consultation
- **Municipal pre-consultation by-laws**

### How to Build It

```
PRE-APPLICATION CONSULTATION PACKAGE

COVER LETTER:
[Brief letter requesting a pre-consultation meeting,
describing the proposal, and listing enclosed materials]

REQUIRED MATERIALS:

1. COMPLETED PRE-CONSULTATION REQUEST FORM
   [Municipal form — varies by municipality]
   Key fields:
   - Property address and legal description
   - Current use and zoning
   - Official Plan designation
   - Proposal description
   - Preliminary development statistics
   - Applicant/agent contact information

2. PRELIMINARY CONCEPT PLAN
   [Does not need to be final — conceptual is acceptable]
   Include:
   - Site plan showing building location on lot
   - Floor plan (schematic)
   - Building height and massing
   - Proposed setbacks
   - Parking and access concept
   - Landscaping concept

3. PRELIMINARY PLANNING ANALYSIS
   [Brief version — 3-5 pages]
   - Zoning analysis (compliance matrix draft)
   - Identification of known variances/amendments needed
   - Preliminary policy analysis
   - Known site constraints

4. CONTEXT MAP
   - Aerial photo showing subject property and surroundings
   - Walking radius to transit, parks, schools, amenities
   - Land use context

5. PRELIMINARY DEVELOPMENT DATA
   | Metric | Value |
   |--------|-------|
   | Lot area | X sq m |
   | Proposed GFA | X sq m |
   | Proposed height | X m / Y storeys |
   | Proposed units | X |
   | Proposed parking | X |
   | Estimated FSI | X |

WHAT TO ASK AT THE MEETING:
1. What application type do you recommend? (MV, ZBA, OPA?)
2. What studies will be required?
3. Are there any site-specific issues we should be aware of?
4. What is the current processing timeline?
5. Are there any upcoming policy changes that could affect
   this application?
6. Who will be the assigned planner?
7. Are there any design concerns at this stage?
8. Is there a community consultation requirement?
9. What is the fee structure?
10. Are there any municipal programs or incentives
    we should know about? (e.g., affordable housing
    incentives, development charge deferrals)

AFTER THE MEETING:
- Staff will issue a pre-consultation response letter
  listing all required materials
- Use this letter as the definitive checklist for
  preparing the complete application
- Response typically takes 2-4 weeks
- Pre-consultation meeting notes should be retained
  for the file
```
